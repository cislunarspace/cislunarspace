# AI 问答页面现代化重构

## 目标
将 AiChat.vue 重构为现代化的大模型对话界面，靠近 ChatGPT/Claude/DeepSeek 的视觉风格和交互体验。

## 任务列表

- [x] 1. 添加侧边栏组件（对话历史管理，localStorage 持久化）
- [x] 2. 实现深色/浅色主题切换（跟随系统 + 手动切换）
- [x] 3. 添加流式输出光标动画
- [x] 4. 现代化输入框设计（圆角、阴影、发送按钮样式）
- [x] 5. 精致的头像（SVG 图标替代字母）
- [x] 6. 代码块样式优化
- [x] 7. 三点加载动画（替代文字 "AI 正在输入..."）
- [x] 8. 现代化欢迎页设计（带图标的分类建议）
- [x] 9. 自定义滚动条样式
- [x] 10. 响应式优化（移动端适配）
- [x] 11. 平滑过渡动画（主题切换、消息出现）
- [x] 12. 英文版国际化更新

## 技术方案
- 样式方案：Scoped CSS + CSS 变量（支持主题切换）
- 图标：内联 SVG（无外部依赖）
- 代码高亮：highlight.js（CDN 或 npm）
- 对话历史：localStorage 持久化
- 主题：CSS 变量 + `prefers-color-scheme` 媒体查询

## 备注
- 保持现有 API 集成和核心逻辑不变
- 确保向后兼容 i18n
- 不破坏现有论坛等页面
