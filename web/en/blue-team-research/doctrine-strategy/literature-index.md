# Literature Index & Extract Rules

## Index fields

Each entry includes at minimum: title, author or agency, publication date, access (URL or library record), language, and topic tags (strategy / doctrine / policy, etc.).

## Extract rules

Quotes preserve original wording with page or section numbers; translated text must cite the translation source. Separate revisions of the same title get separate entries with supersession notes.

## Versions

The index is updated annually or at major policy milestones; broken links record the last observed date and archive location if any.
