# Search & Metadata

## Metadata fields

Each document or attachment should include: title, author or editor, publication date, last revision, topic tags (track), and public-scope statement.

## Search facets

Filter by keyword, date range, document type (doctrine / report / program file, etc.), and source organization; full-text and abstract search in parallel.

## Updates

Revisions carry version or revision date; major changes summarize in the overview.
