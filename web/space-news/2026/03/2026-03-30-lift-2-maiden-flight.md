# 力箭二号首飞成功：中国首款 CBC 构型火箭入轨，商业航天进入大运力时代

**摘要：** 2026年3月30日19时00分，中科宇航力箭二号遥一运载火箭「国际纺都号」在东风商业航天创新试验区成功首飞，将新征程01卫星、新征程02卫星（轻舟初样试飞船）和天视卫星01星精准送入预定轨道。这是我国首款「通用助推器核心」（CBC）构型的运载火箭，标志着中国商业航天正式进入大运力、低成本、可重复使用的新阶段。

## 火箭概况

力箭二号由中科宇航研制，是我国首款采用通用助推器核心（Common Booster Core, CBC）构型的运载火箭：

| 参数 | 数值 |
|------|------|
| 总长 | 53 米 |
| 芯级直径 | 3.35 米 |
| 整流罩直径 | 4.2 米 |
| 一子级含助推器直径 | 5.8 米 |
| 起飞重量 | 625 吨 |
| 起飞推力 | 753 吨 |

CBC构型的核心优势在于模块化设计——芯级与助推器在外观和结构上高度统一，如同「拼积木」般灵活组合，显著降低制造成本和生产周期。

## 首飞任务

本次首飞搭载三颗卫星：
- **新征程01卫星**
- **新征程02卫星**（即轻舟初样试飞船）
- **天视卫星01星**

首飞即服务国家重大战略和重大工程建设，是力箭系列运载火箭的第12次发射。

## 未来展望

力箭二号总指挥杨浩亮表示，此次首飞是我国新一代天地货物运输体系建设的重要信号。后续力箭二号将开展百公里级回收试验，成熟后将为星网工程提供稳定可靠的发射服务。

中科宇航同时披露了各线产品时间表：
- **力箭二号重型**：计划2028年首飞
- **力箭三号**：计划2030年首飞
- **力鸿二号**（可重复使用运载器）：计划2026年底至2027年一季度首飞

力箭二号将支撑低轨通信卫星星座组网、低成本空间站货运、中高轨卫星发射及深空探测等多元化任务需求。
