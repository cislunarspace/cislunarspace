# 我国成功发射试验三十三号卫星

**摘要：** 北京时间 **2026 年 3 月 27 日 12 时 11 分**，我国在 **酒泉卫星发射中心** 使用 **长征二号丙运载火箭/远征一号S上面级**，成功将 **试验三十三号卫星** 发射升空，卫星顺利进入预定轨道，发射任务取得圆满成功。该卫星主要用于 **空间环境探测科学试验**。此次任务是 **长征系列运载火箭的第 635 次飞行**。

## 任务概况

| 项目 | 参数 |
|------|------|
| 运载火箭 | 长征二号丙运载火箭 / 远征一号S上面级 |
| 发射场 | 酒泉卫星发射中心 |
| 发射时间（北京时间） | 2026 年 3 月 27 日 12:11 |
| 卫星名称 | 试验三十三号卫星 |
| 任务性质 | 空间环境探测科学试验 |
| 轨道 | 预定轨道 |
| 长征系列火箭飞行次数 | 第 635 次 |

## 背景信息

试验三十三号卫星是由中国科学院牵头研制的科学试验卫星，主要用于空间环境探测相关技术验证和科学实验。远征一号S上面级是由中国航天科技集团研制的全新上面级产品，可在一次任务中将卫星精确送入多个不同轨道，提高了太阳同步轨道任务的适应性和灵活性。
