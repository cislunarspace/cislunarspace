# 作战应用与聚焦场景 · 概述

## 研究内容

依据公开报道与学术文献，分析冲突或演习中太空能力之运用方式，包括但不限于太空支援、遥感、通信、导航与对抗等议题之公开论述。

题目范围过大时，收缩至明确场景，例如：轨道对抗相关公开概念与论述；杀伤链某一环节的公开技术或运用描述；限定地域与时段内的公开战例分析。

## 分析方法

每一研究单元须包含：问题定义、文献与报道来源、结论与存疑之点。与条令、装备章节对照，检验「文本—能力—运用」链条是否一致。

## 成果形式

| 类别 | 内容 |
|------|------|
| 场景说明 | 时间、地域、参与方、太空相关要素（均须公开来源） |
| 证据整理 | 文献与报道引用及矛盾点标注 |
| 对照分析 | 条令概念、装备能力与战场观察之对应关系 |
