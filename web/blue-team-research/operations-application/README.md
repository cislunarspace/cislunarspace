# 作战应用与聚焦场景 · 概述

## 研究内容

依据公开报道与学术文献，分析冲突或演习中太空能力之运用方式。建议分题：**地区冲突中的运用**（如俄乌、中东等公开报道）、**收窄场景**（轨道对抗、杀伤链闭环），并与装备技术侧公开描述相互对照。子页见 [地区冲突中的太空运用](./regional-conflicts)、[轨道对抗聚焦](./orbital-confrontation)、[杀伤链闭环聚焦](./kill-chain-focus)。

题目范围过大时，须收缩至明确场景或单一环节，避免泛泛而谈。

## 分析方法

每一研究单元须包含：问题定义、文献与报道来源、结论与存疑之点。与条令、装备章节对照，检验「文本—能力—运用」链条是否一致。

## 成果形式

| 类别 | 内容 |
|------|------|
| 场景说明 | 时间、地域、参与方、太空相关要素（均须公开来源） |
| 证据整理 | 文献与报道引用及矛盾点标注 |
| 对照分析 | 条令概念、装备能力与战场观察之对应关系 |
