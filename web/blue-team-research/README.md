# 蓝军研究

## 研究范围

本模块收录基于**公开来源**的蓝军研究资料整理与论述，材料限于合法、可公开发布与引用的文献及报道，不涉及涉密内容。

## 研究宗旨与目标

蓝军研究以提升对对手太空作战体系之**系统认知**为基本任务，持续跟踪太空作战领域公开信息动态，形成可复核、可更新的认识基础。研究活动须与国家太空作战力量战斗力生成需求相衔接，侧重分析能力形成路径、运用逻辑与可对照的技术与管理环节。

对公开条令、战略文件、装备体系与典型运用方式加以整理与归纳，支撑研究选题分解、研究路径设计及阶段性成果评估。

### 目标层级

| 层级 | 内涵 |
|------|------|
| 战略愿景 | 人类太空探索与长期生存空间拓展 |
| 国家航天 | 航天科技与工程能力的持续提升与贡献 |
| 战斗力建设 | 军队太空相关战斗力生成中的认知、方法与手段支撑 |

蓝军研究作为上述层级之**情报与认识基础**，与相关工程技术研究相互补充。

## 研究方向划分

| 方向 | 研究对象 |
|------|----------|
| 条令与战略 | 公开战略文件、条令体系及「政策—行动」对应关系 |
| 装备与技术 | 体系架构、型号与项目、商业及产业参与（公开信息） |
| 作战应用 | 公开战例、演习或限定场景（如轨道对抗、杀伤链环节） |

选题须控制范围：主题过大时应收缩为可完成的子问题，以保证论述深度与资料可穷尽性。

## 阶段性工作

| 阶段 | 内涵 |
|------|------|
| 文献与索引 | 建立可复核的书目、术语与摘录基础 |
| 专题深化 | 在单一方向或交叉议题上形成结构化论述 |
| 成果固化 | 形成可引用的文稿、图表与更新记录，并通过本站发布 |
