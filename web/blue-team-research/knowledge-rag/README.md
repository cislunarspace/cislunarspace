# 资料共享与知识库 · 概述

## 资料管理

蓝军研究各章节成果统一发布于本站，采用结构化目录与统一引用格式；元数据、命名与版本管理遵循本站编辑规范。

## 检索服务

建设面向全文与摘要的检索能力，支持按主题、时间与来源筛选。

## 知识服务（规划）

在合规前提下，分阶段实施检索与增强式问答（RAG）等能力；实施须符合国家保密、数据安全及适用行业监管要求。
