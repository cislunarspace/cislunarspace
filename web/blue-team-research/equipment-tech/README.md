# 装备与技术发展 · 概述

## 研究内容

在公开信息范围内，跟踪太空作战相关**体系结构**、**型号与项目**、**研制与部署进程**及**商业航天参与方式**，形成可核对的技术与管理描述。专题入口见 [太空发展局（SDA）与相关方案](./sda-programs)、[SpaceX 与产业参与](./spacex-role)。

## 技术路线

信息组织采用「体系—项目—技术要点」分层；技术判断须标注来源与日期，区分官方发布、合同公开信息与媒体报道。
