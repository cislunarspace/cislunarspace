## [地月空间知识体系](/知识体系)

系统梳理地月空间探索的五大核心支柱：空间环境特征、限制性三体问题动力学、任务轨道架构设计、高效转移轨道规划、制导导航与控制技术。内容涵盖从基础概念到前沿研究的完整知识链条，帮助不同背景的读者建立结构化认知框架。

## [学习路径规划](/学习路径)

面向三类受众的定制化学习路线：
- **爱好者入门**：图文并茂的科普内容，解读拉格朗日点、晕轨道等核心概念，结合CAPSTONE、阿耳忒弥斯等任务案例激发兴趣
- **学生进阶**：从经典力学到数值仿真，提供数学基础、轨道力学、MATLAB/Python工具链的系统训练
- **专业深耕**：聚焦高保真模型、多体摄动分析、自主GNC算法等前沿课题，支持科研与工程实践

## [轨道动力学解析](/动力学)

深入剖析限制性三体问题下的关键动力学现象：
- 拉格朗日点与平动点轨道的几何特性与稳定性分析
- 不变流形（稳定/不稳定流形）的物理意义与工程应用
- 弱稳定性边界（WSB）理论与低能量转移的混沌机制
- 数值计算方法：多shooting法、微分修正、庞加莱截面分析

## [任务案例与实践](/案例实践)

通过真实任务还原轨道设计的工程逻辑：
- **CAPSTONE任务**：近直线晕轨道（NRHO）的验证与维持策略
- **GRAIL/SMART-1**：低能量转移轨道的实际应用与燃料优化
- **阿耳忒弥斯计划**：月球门户空间站的轨道架构与交通系统构想
- **LONEStar实验**：自主光学导航在深空环境中的技术验证

## [技术术语词典](/技术词典)

地月空间领域专业术语权威释义库，覆盖：
- 动力学基础：限制性三体问题、雅可比常数、零速度面
- 轨道类型：晕轨道、李雅普诺夫轨道、远距离逆行轨道（DRO）
- 转移策略：霍曼转移、弱稳定性边界转移、低推力螺旋
- GNC技术：自主导航、视觉里程计、模型预测控制
- 任务要素：轨道维持、交会对接、深空测控
