## 地月空间概念

地月空间（Cislunar Space）是指地球与月球之间的空间区域，包括地球同步轨道以外的所有空间。这一区域正成为21世纪太空探索和开发的新前沿，具有重要的科学、技术和战略价值。

### 空间环境特征
- **引力环境**：地球、月球和太阳引力的复杂相互作用
- **辐射环境**：范艾伦辐射带、太阳宇宙射线、银河宇宙射线
- **等离子体环境**：地球磁层、太阳风、月球尾迹
- **微流星体环境**：流星体通量分布和撞击风险

### 限制性三体问题动力学
- **圆形限制性三体问题（CR3BP）**：基础理论模型
- **雅可比积分与零速度面**：能量约束分析
- **拉格朗日点**：五个平动点的位置与稳定性
- **周期轨道族**：晕轨道、李雅普诺夫轨道、垂直轨道

### 任务轨道架构设计
- **近直线晕轨道（NRHO）**：月球门户空间站首选轨道
- **远距离逆行轨道（DRO）**：长期稳定存储轨道
- **低能量转移轨道**：基于流形理论的节能转移
- **轨道维持策略**：摄动补偿和轨道控制

### 高效转移轨道规划
- **弱稳定性边界（WSB）理论**：混沌动力学在轨道设计中的应用
- **不变流形**：稳定流形与不稳定流形的计算与应用
- **多体摄动轨道**：考虑太阳引力的高精度轨道设计
- **优化算法**：直接法、间接法、启发式算法

### 制导导航与控制技术
- **自主导航**：光学导航、X射线脉冲星导航、相对导航
- **制导算法**：预测校正制导、最优制导、自适应制导
- **控制技术**：模型预测控制、滑模控制、鲁棒控制
- **故障检测与处理**：系统健康管理、容错控制
