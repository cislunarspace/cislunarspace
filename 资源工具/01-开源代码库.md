## CR3BP数值积分

### 简介
圆形限制性三体问题（CR3BP）是研究地月空间轨道动力学的基础模型。本节提供CR3BP数值积分的开源代码实现。

### 代码示例
```python
import numpy as np
from scipy.integrate import solve_ivp

def cr3bp_equations(t, state, mu):
    """
    圆形限制性三体问题动力学方程
    """
    x, y, z, vx, vy, vz = state
    
    # 到两个主天体的距离
    r1 = np.sqrt((x + mu)**2 + y**2 + z**2)
    r2 = np.sqrt((x - 1 + mu)**2 + y**2 + z**2)
    
    # 加速度项
    ax = 2*vy + x - (1-mu)*(x+mu)/r1**3 - mu*(x-1+mu)/r2**3
    ay = -2*vx + y - (1-mu)*y/r1**3 - mu*y/r2**3
    az = -(1-mu)*z/r1**3 - mu*z/r2**3
    
    return [vx, vy, vz, ax, ay, az]

# 使用示例
mu = 0.01215  # 地月系统质量参数
initial_state = [0.8, 0, 0, 0, 0.5, 0]  # 初始状态
t_span = [0, 10]  # 积分时间范围

sol = solve_ivp(cr3bp_equations, t_span, initial_state, 
                args=(mu,), method='RK45', rtol=1e-12, atol=1e-12)
```

### 功能特性
- 支持多种积分器（RK45, DOP853, Radau等）
- 可配置积分精度和步长
- 提供事件检测功能（如穿越平面事件）
- 支持并行计算加速

### 应用场景
- 晕轨道生成与稳定性分析
- 流形计算与转移轨道设计
- 庞加莱截面分析
- 长期轨道演化研究

## 流形计算

### 简介
稳定流形和不稳定流形是连接不同轨道区域的关键结构，在低能量转移轨道设计中具有重要应用。

### 计算方法
1. **线性近似法**：在周期轨道附近线性化动力学方程
2. **打靶法**：通过数值积分和边界条件迭代
3. **庞加莱映射法**：在截面空间中追踪流形

### 代码实现
```python
def compute_manifold(periodic_orbit, mu, direction='stable', num_trajectories=10):
    """
    计算周期轨道的流形
    """
    # 获取周期轨道的单值矩阵
    monodromy_matrix = compute_monodromy(periodic_orbit, mu)
    
    # 计算特征值和特征向量
    eigenvalues, eigenvectors = np.linalg.eig(monodromy_matrix)
    
    # 选择稳定或不稳定特征向量
    if direction == 'stable':
        idx = np.argmin(np.abs(eigenvalues))
    else:
        idx = np.argmax(np.abs(eigenvalues))
    
    manifold_direction = eigenvectors[:, idx]
    
    # 生成流形轨迹
    trajectories = []
    for i in range(num_trajectories):
        # 在特征方向添加小扰动
        perturbation = 1e-6 * manifold_direction * (i - num_trajectories/2)
        initial_state = periodic_orbit[0] + perturbation
        
        # 正向或反向积分
        if direction == 'stable':
            t_span = [0, -50]  # 反向积分
        else:
            t_span = [0, 50]   # 正向积分
            
        sol = solve_ivp(cr3bp_equations, t_span, initial_state, 
                       args=(mu,), method='RK45')
        trajectories.append(sol)
    
    return trajectories
```

## 轨道可视化模板

### 简介
提供多种轨道可视化模板，支持2D和3D展示，便于分析和演示。

### 可视化类型
1. **3D轨道图**：展示轨道在三维空间中的形态
2. **相空间图**：展示状态空间中的轨道演化
3. **庞加莱截面**：展示轨道在截面上的交点
4. **能量-角动量图**：展示轨道能量特性

### 代码模板
```python
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

def plot_3d_trajectory(trajectory, title="轨道轨迹"):
    """
    绘制3D轨道图
    """
    fig = plt.figure(figsize=(10, 8))
    ax = fig.add_subplot(111, projection='3d')
    
    # 绘制主天体位置
    ax.scatter([-0.01215], [0], [0], s=100, c='blue', label='地球')
    ax.scatter([0.98785], [0], [0], s=50, c='gray', label='月球')
    
    # 绘制轨道
    ax.plot(trajectory[:, 0], trajectory[:, 1], trajectory[:, 2], 
            'b-', linewidth=1, label='轨道')
    
    # 设置坐标轴
    ax.set_xlabel('X (归一化距离)')
    ax.set_ylabel('Y (归一化距离)')
    ax.set_zlabel('Z (归一化距离)')
    ax.set_title(title)
    ax.legend()
    ax.grid(True)
    
    plt.tight_layout()
    return fig, ax
```

### 交互功能
- 支持旋转、缩放、平移
- 可添加多个轨道对比
- 支持动画演示
- 可导出高质量图片和视频
