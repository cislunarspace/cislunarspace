## GMAT在地月任务中的应用教程

### GMAT简介
通用任务分析工具（GMAT）是NASA开发的开源轨道设计和任务分析软件，广泛应用于深空探测任务。

### 安装与配置

#### 系统要求
- **操作系统**：Windows 10/11, macOS 10.14+, Linux
- **内存**：8GB RAM（推荐16GB）
- **存储空间**：2GB可用空间
- **Java**：JDK 11或更高版本

#### 安装步骤
1. **下载GMAT**
   ```bash
   # 从官方GitHub仓库下载最新版本
   https://github.com/NASA-AMMOS/GMAT/releases
   ```

2. **安装Java环境**
   ```bash
   # 下载并安装OpenJDK
   https://adoptium.net/
   ```

3. **配置环境变量**
   ```bash
   # Windows系统
   set JAVA_HOME=C:\Program Files\Eclipse Adoptium\jdk-11.0.15.10-hotspot
   set PATH=%JAVA_HOME%\bin;%PATH%
   
   # Linux/macOS系统
   export JAVA_HOME=/usr/lib/jvm/java-11-openjdk
   export PATH=$JAVA_HOME/bin:$PATH
   ```

### 基础教程

#### 创建第一个地月任务
1. **新建任务**
   - 打开GMAT，点击"File" → "New Mission"
   - 设置任务名称为"LunarTransfer"

2. **添加天体**
   ```matlab
   % 在脚本编辑器中添加
   Create Earth Earth
   Create Moon Moon
   Create LSpacecraft Spacecraft
   
   % 设置地球参数
   Earth.EphemerisSource = 'DE405'
   Earth.CentralBody = 'Sun'
   
   % 设置月球参数
   Moon.EphemerisSource = 'DE405'
   Moon.CentralBody = 'Earth'
   ```

3. **定义轨道**
   ```matlab
   % 设置航天器初始轨道
   LSpacecraft.Epoch = '1 Jan 2025 00:00:00.000'
   LSpacecraft.CoordinateSystem = 'EarthMJ2000Eq'
   LSpacecraft.DisplayStateType = 'Keplerian'
   
   % 地球停泊轨道
   LSpacecraft.SMA = 6678.137  % km
   LSpacecraft.ECC = 0.0
   LSpacecraft.INC = 28.5      % deg
   LSpacecraft.RAAN = 0.0      % deg
   LSpacecraft.AOP = 0.0       % deg
   LSpacecraft.TA = 0.0        % deg
   ```

#### 地月转移轨道设计
```matlab
% 定义地月转移机动
Create ImpulsiveBurn TLI

TLI.CoordinateSystem = 'EarthMJ2000Eq'
TLI.Origin = Earth
TLI.Axes = 'VNB'

% 设置转移机动参数
TLI.Element1 = 3100  % ΔV in m/s
TLI.Element2 = 0
TLI.Element3 = 0

% 执行机动
BeginMissionSequence
   Propagate LSpacecraft(Earth) {LSpacecraft.ElapsedDays = 0.5}
   Maneuver TLI(LSpacecraft)
   Propagate LSpacecraft(Earth,Moon) {LSpacecraft.ElapsedDays = 4.5}
EndMissionSequence
```

### 高级功能

#### 优化地月转移轨道
```matlab
% 使用微分修正优化转移轨道
Create DifferentialCorrector DC1

DC1.MaximumIterations = 50
DC1.ReportStyle = 'Normal'
DC1.ReportFile = 'DC_Report.txt'

% 定义约束条件
Create Constraint ArrivalConstraint

ArrivalConstraint.Type = 'OrbitState'
ArrivalConstraint.Spacecraft = LSpacecraft
ArrivalConstraint.CentralBody = Moon
ArrivalConstraint.Element1 = 'SMA'
ArrivalConstraint.Desired = 1838  % 100km月球轨道高度
ArrivalConstraint.Tolerance = 1.0

% 执行优化
DC1.AddVariable(LSpacecraft.TLI.Element1)
DC1.AddConstraint(ArrivalConstraint)
DC1.Solve()
```

#### 多体摄动分析
```matlab
% 添加摄动力模型
Create ForceModel HighFidelityFM

HighFidelityFM.CentralBody = Earth
HighFidelityFM.PointMasses = {Sun, Moon, Jupiter, Saturn}
HighFidelityGM.Drag = 'None'
HighFidelityFM.SRP = 'Spherical'

% 使用高保真力模型传播
Create Propagator HighFidelityProp

HighFidelityProp.FM = HighFidelityFM
HighFidelityProp.Type = 'RungeKutta89'
HighFidelityProp.InitialStepSize = 60
HighFidelityProp.Accuracy = 1e-12

Propagate LSpacecraft(HighFidelityProp) {LSpacecraft.Epoch = '5 Jan 2025 00:00:00.000'}
```

### 实用技巧

#### 脚本自动化
```matlab
% 批量处理多个任务
for i = 1:10
    % 修改初始条件
    LSpacecraft.SMA = 6678.137 + (i-1)*100
    
    % 执行任务
    Propagate LSpacecraft(Earth) {LSpacecraft.ElapsedDays = 0.5}
    Maneuver TLI(LSpacecraft)
    Propagate LSpacecraft(Earth,Moon) {LSpacecraft.ElapsedDays = 4.5}
    
    % 保存结果
    ReportFile = sprintf('Result_%d.txt', i)
    WriteReport(LSpacecraft, ReportFile)
end
```

## Orekit在地月任务中的应用教程

### Orekit简介
Orekit是欧洲空间局支持的开源天体动力学库，基于Java开发，提供丰富的轨道力学算法。

### Python环境配置
```python
# 安装Orekit Python绑定
pip install orekit
pip install jpype1

# 下载Orekit数据包
import orekit
orekit.initVM()

from orekit.pyhelpers import setup_orekit_curdir
setup_orekit_curdir()
```

### 基础轨道计算
```python
from org.orekit.bodies import CelestialBodyFactory, OneAxisEllipsoid
from org.orekit.frames import FramesFactory
from org.orekit.orbits import KeplerianOrbit
from org.orekit.time import AbsoluteDate, TimeScalesFactory
from org.orekit.utils import Constants, IERSConventions

# 创建地月系统
earth = CelestialBodyFactory.getEarth()
moon = CelestialBodyFactory.getMoon()

# 创建地球参考椭球体
earthFrame = FramesFactory.getITRF(IERSConventions.IERS_2010, True)
earthShape = OneAxisEllipsoid(Constants.WGS84_EARTH_EQUATORIAL_RADIUS,
                              Constants.WGS84_EARTH_FLATTENING,
                              earthFrame)

# 定义初始轨道
initialDate = AbsoluteDate(2025, 1, 1, 0, 0, 0.0, TimeScalesFactory.getUTC())
position = [6678137.0, 0.0, 0.0]  # 位置矢量 (m)
velocity = [0.0, 7668.63, 0.0]    # 速度矢量 (m/s)
inertialFrame = FramesFactory.getEME2000()

orbit = KeplerianOrbit(position, velocity, inertialFrame, initialDate, 
                       earth.getGM())
```

### 地月转移仿真
```python
from org.orekit.propagation import SpacecraftState
from org.orekit.propagation.analytical import KeplerianPropagator
from org.orekit.propagation.numerical import NumericalPropagator
from org.orekit.forces import gravity

# 创建航天器状态
initialState = SpacecraftState(orbit, 1000.0)  # 1000kg质量

# 创建数值传播器
propagator = NumericalPropagator(orbit)

# 添加地球引力
earthAttraction = gravity.NewtonianAttraction(earth.getGM())
propagator.addForceModel(earthAttraction)

# 添加月球第三体摄动
moonAttraction = gravity.ThirdBodyAttraction(moon)
propagator.addForceModel(moonAttraction)

# 传播轨道
finalDate = initialDate.shiftedBy(5.0 * 86400.0)  # 5天后
finalState = propagator.propagate(finalDate)
```

## STK在地月任务中的应用教程

### STK简介
Systems Tool Kit（STK）是AGI公司开发的商业级任务分析软件，提供强大的可视化分析和系统仿真能力。

### 快速入门

#### 创建地月场景
1. **新建场景**
   ```javascript
   // STK Connect命令
   New / Scenario LunarMission
   SetTimePeriod * "1 Jan 2025 00:00:00.000" "10 Jan 2025 00:00:00.000"
   ```

2. **添加地月系统**
   ```javascript
   // 添加地球和月球
   VO * CentralBody Earth
   VO * CentralBody Moon
   
   // 设置月球轨道
   SetDefault * CentralBody/Moon EphemerisSource "DE405"
   ```

3. **创建航天器**
   ```javascript
   New / */Satellite LunarSat
   SetState */Satellite/LunarSat Classical TwoBody "1 Jan 2025 00:00:00.000" 6678.137 0.0 28.5 0.0 0.0 0.0 Earth
   ```

#### 地月转移分析
```javascript
// 定义转移轨道
MCS */Satellite/LunarSat
SetValue MainSequence.Transfer.Type "Lunar"
SetValue MainSequence.Transfer.Departure.Orbit.Epoch "1 Jan 2025 00:00:00.000"
SetValue MainSequence.Transfer.Departure.Orbit.SemiMajorAxis 6678.137
SetValue MainSequence.Transfer.Arrival.Orbit.SemiMajorAxis 1838.0
SetValue MainSequence.Transfer.Arrival.Orbit.Inclination 90.0

// 计算转移轨道
Compute */Satellite/LunarSat MainSequence
```

### 高级分析功能

#### 覆盖分析
```javascript
// 创建地面站
New / */Facility GroundStation
SetPosition */Facility/GroundStation Geodetic 39.9 116.4 0.0

// 分析覆盖
Access */Satellite/LunarSat */Facility/GroundStation
ReportCreate */Satellite/LunarSat Type "Access" Style "Standard" File "CoverageReport.txt"
```

#### 通信链路分析
```javascript
// 创建通信系统
New / */Transmitter Tx
New / */Receiver Rx

// 配置链路参数
SetValue */Transmitter/Tx Frequency 2.2e9
SetValue */Transmitter/Tx Power 20
SetValue */Receiver/Rx GOverT 20

// 分析链路预算
Chain */Satellite/LunarSat/Tx */Facility/GroundStation/Rx
Compute */Chain/Chain1
```
